transition学习示例

学习自该文章： https://www.cnblogs.com/xiaohuochai/p/5347930.html

定义

	.test {
		height: 100px;
		width: 100px;
		background-color: #00ff00;
		transition-duration: 3s;
		transition-property: all;
		transition-timing-function: ease;
		transition-delay: 0s;
	}
	.test:hover {
		width: 500px;
	}

复合属性

	.test {
		height: 100px;
		width: 100px;
		background-color: #00ff00;
		/*代表持续时间为2s，延迟时间为默认值0s*/
		transition: 2s;
	}

	.test:hover {
		width: 500px;
	}
	
	.test {
		height: 100px;
		width: 100px;
		background-color: #00ff00;
		/*代表持续时间为1s，延迟时间为2s*/
		transition: 1s 2s;
	}

	.test:hover {
		width: 500px;
	}

过渡属性transition-property

	值: none | all | <transition-property>[,<transition-property>]*

	初始值: all

	应用于: 所有元素

	继承性: 无

过渡持续时间transition-duration

	值: <time>[,<time>]*

	初始值: 0s

	应用于: 所有元素

	继承性: 无
	
	.test {
		height: 100px;
		width: 100px;
		background-color: #00ff00;
		/*代表持续时间为1s，延迟时间为2s*/
		transition: 1s 2s;
	}

	.test:hover {
		width: 500px;
	}
