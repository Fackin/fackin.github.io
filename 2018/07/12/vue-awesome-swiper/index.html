api：中文api - Swiper中文网
官方demo：vue-awesome-swiper
官方配置：GitHub - surmon-china/vue-awesome-swiper: 🏆 Swiper component for @vuejs
下载包
npm install vue-awesome-swiper --save
或yarn方式
yarn add vue-awesome-swiper --save

全局引用
import方式

也可以require方式

Vue.use(VueAwesomeSwiper)
