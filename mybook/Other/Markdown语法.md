## 8.表格

```
表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容

第二行分隔表头和内容，- 一个就行，为了对齐多个
默认左对齐
:--: 中间对齐
--: 右对齐
注：原生的语法两边都要用 | 包起来
```

| 表头 | 表头 | 表头 |
| ---- | :--: | ---: |
| 内容 | 内容 | 内容 |
| 内容 | 内容 | 内容 |
